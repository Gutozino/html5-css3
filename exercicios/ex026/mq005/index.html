Entendendo o design responsivo

Para tentar resumir um pouco, design web responsivo é o forma de uso do HTML e CSS para a criação de uma aplicação (ou página) web que irá funcionar corretamente em todos os dispositivos, independentemente do tamanho da tela de cada dispositivo. Todos os elementos que temos em tela, como botões, imagens, funcionalidade e demais recursos devem ser vistos e ter a mesma funcionalidade em monitores gigantescos e também em telas menores como a de um smartphone. E é esse o pensamento de responsividade que todos nós, desenvolvedores de softwares, devemos ter ao criar qualquer tipo de página ou aplicação web. É válido salientar também, que existem diversas técnicas além do Media Query que podemos utilizar para nos auxiliar no desenvolvimento de páginas responsivas. Usar unidades relativas ao invés de absolutas (por exemplo, usar % ao invés de px); Configurar corretamente o viewport/scale da página. Utilizar recursos de acessibilidade de páginas. Porém, neste artigo vamos focar em Media Query pois na minha opinião é a melhor maneira de começarmos.

O que são Media Queries

Media Query é um recurso do CSS que veio juntamente com a versão CSS3. Com este recurso, desenvolvedores tem a possibilidade de modificar o comportamento da página de acordo com o tamanho (widht e height) das telas, ou seja, mediante a diferentes tamanhos de telas, a página terá diferentes comportamentos. Utilizando Media Query você terá acesso a uma espécie de Framework para escrever o código somente uma vez e utiliza-lo em toda a sua aplicação/página. Logicamente, se estamos falando somente de uma página em específico, talvez você não veja tanta diferença assim, mas no momento que você trabalhar em uma aplicação que controla dezenas de páginas, você verá uma mudança gigantesca no uso do Media Query. Essa fato se dá pois ao usar Media Query você estará levando em consideração alguns fatores, como mencionei widht e height, mas muito além disso, você deverá levar em consideração a estrutura, localização, links, posições, dentre outros.
